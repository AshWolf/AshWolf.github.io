BBB Simulator:
A VR project featuring multiple interactable objects and NPCs, along with a minigame where the player tries to teleport away from a monster for as long as possible. This was created using the Unreal 5 engine, utilizing its Blueprint/Event Graph system/language. It was created over the course of three weeks in a team of two, utilizing an agile workflow and using the project management software Jira.


A2_Go:
A Pokemon Go-inspired AR game that involves visiting various landmarks of Ann Arbor, while collecting new seeds, planting trees, and fending off various different types of squirrels. This was created using Unity's AR template, and utilizes Map Box for the world map and Mixamo for animation. It was created over the course of three and a half weeks in a team of two, utilizing an agile workflow and using the project management software Jira.

Aesthetics to be improved when I have the time.
