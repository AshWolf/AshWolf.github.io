VR Exercise + Cognitive Game
Unreal Engine
Project Overview
A VR game that combines physical and mental fitness, letting players work out physically and mentally while having fun. NeuroFit Fusion features 4 different minigames, 3 of which I designed and implemented. It was created in under four weeks, using Unreal Engine, in a team of five, utilizing an agile workflow and using the project management software Jira.
